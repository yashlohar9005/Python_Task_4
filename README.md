Loops & Iterations Program (Python)

----------------------------------------------------------------------


Author: Yash Lohar

A simple Python program to demonstrate the use of loops in Python including for loop, while loop, break, continue, and practical examples.

This project is part of TASK: Loops & Iteration and demonstrates how iteration and automation work in Python.

----------------------------------------------------------------------


Tools Used

  • Python
  
  • VS Code

----------------------------------------------------------------------

Features

  • Prints numbers from 1 to 100 using for loop
  
  • Countdown timer using while loop
  
  • Demonstrates break and continue
  
  • Iterates over characters in a string
  
  • Generates multiplication table of 2
  
  • Uses range() with steps (even numbers)
  
  • Combines loop with conditions (odd numbers)
  
  • Real-world example: Shopping cart total calculation
